# OpenStreetMap Data

**Source:** Geofabrik / BBBike  
**Download Date:** February 10, 2026  
**URL:** https://download.geofabrik.de/asia/india.html  
**License:** ODbL (Open Database License)  
**Data Year:** 2024-2026 (continuously updated)

## Files

- `roads_major.shp` - Motorways, highways, primary roads
- `settlements.shp` - Cities, towns with population >1000

## Attributes

**Roads:**
- `highway` - Road type (motorway, primary, secondary)
- `name` - Road name
- `ref` - Reference number (NH1, SH12)

**Settlements:**
- `name` - Place name
- `place` - Type (city, town, village)
- `population` - Population estimate

## Usage

- Human disturbance index calculation
- Road density near reserves
- Settlement proximity analysis
- Context for Story Map
