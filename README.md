# Tiger Conservation Success Stories: India's Protected Areas (2006-2022)

An ArcGIS Story Map analyzing spatial and temporal patterns of tiger population recovery in India's national parks and tiger reserves.

## ğŸ… Project Overview

This project examines tiger conservation success in India by mapping population changes across major protected areas from 2006 to 2022. Using official census data and spatial analysis, it identifies reserves with the strongest recovery trends and highlights effective conservation strategies.
